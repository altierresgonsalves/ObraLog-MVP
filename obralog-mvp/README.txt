ObraLog MVP
===========

Como abrir o aplicativo:
------------------------
Este projeto foi desenvolvido utilizando tecnologias web padrão (HTML, CSS e JavaScript) para garantir compatibilidade e facilidade de execução, dado que o ambiente Node.js não estava disponível.

1. Navegue até a pasta deste projeto.
2. Dê um clique duplo no arquivo 'index.html'.
3. O aplicativo abrirá no seu navegador padrão.

Funcionalidades incluídas:
--------------------------
- Dashboard com lista de obras e progresso visual.
- Linha do tempo (Timeline) para acompanhamento de atividades.
- Design responsivo e tema escuro "Premium".
- Interatividade local (navegação entre telas, modal de criação).

Notas Técnicas:
---------------
- Os dados são mockados (simulados) no arquivo js/app.js para fins de demonstração.
- Não requer instalação de dependências.
